# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
A Machine Learning web application that predicts house prices based on:
- Square Footage
- Number of Bedrooms
- Number of Bathrooms

## 🤖 Algorithm Used
Linear Regression 

## 📊 Dataset
- Source: Kaggle - House Prices Advanced Regression Techniques
- Total Houses: 1460
- Features Used: Square Footage, Bedrooms, Bathrooms

## ✨ Features
- 💰 House Price Prediction
- 🏷️ Price Category (Budget/Mid Range/Luxury)
- 📐 Price Per Square Foot
- 📊 Expected Price Range
- 📏 Affordability Meter
- 🏠 Similar Houses Table
- 📊 Data Analysis & Visualizations

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
- Numpy
- Matplotlib
- Seaborn
